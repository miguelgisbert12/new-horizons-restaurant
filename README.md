🍽️ New Horizons – Página web

New Horizons es una web de una cadena de restaurantes ficticia desarrollada como proyecto final de carrera (TFG). El objetivo del proyecto fue diseñar y construir una experiencia web completa, moderna y responsive utilizando únicamente tecnologías front-end.

👉 Sitio web: [https://new-horizons.netlify.app/](https://new-horizons.netlify.app/)

📌 Sobre el proyecto

Este proyecto simula la presencia digital de una marca de restauración ficticia, incluyendo distintas secciones típicas de una web corporativa de restaurantes.

El foco principal fue trabajar:

Diseño visual atractivo y coherente
Estructura semántica en HTML
Estilos responsive con CSS
Interactividad básica con JavaScript vanilla
Optimización para visualización en distintos dispositivos

🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Diseño responsive (mobile-first)
- Netlify (deploy)

🎯 Objetivos del proyecto

- Crear una web completa desde cero sin frameworks
- Practicar estructuración de proyectos reales
- Mejorar habilidades de diseño UI/UX
- Implementar interactividad básica con JavaScript
- Simular un entorno de cliente real (branding + identidad visual)

📱 Características

- Diseño responsive adaptado a móvil, tablet y desktop
- Página principal con estructura tipo landing
- Secciones informativas (restaurante, menú, contacto, etc.)
- Navegación fluida entre secciones
- Estilo visual consistente y moderno
- Optimización para carga rápida

🧠 Aprendizajes

Durante el desarrollo de este proyecto se trabajaron conceptos como:

- Maquetación con HTML semántico
- Flexbox y/o Grid en CSS
- Organización de estilos en proyectos grandes
- Lógica básica en JavaScript
- Flujo de trabajo completo de un proyecto web
- Despliegue en producción con Netlify
